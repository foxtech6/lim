Codingtest
=========

### Setup
System requirements:
- PHP >=7.0

### How install
- Setup server
- Run ```composer install```

### Usage example
```
$ php bin/console purchase-cigarettes 2 10.00

You bought 2 packs of cigarettes for -9,98€, each for -4,99€.

Your change is:
+-------+-------+
| Coins | Count |
+-------+-------+
| 0.02  | 1     |
+-------+-------+
```
